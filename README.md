# Angular: From Zero to Expert (2026 Edition)

This repository contains my progress, exercises, and projects developed during the **Angular 19+** specialization course. The goal is to master the latest features of the framework and apply modern design patterns.

## Key Technologies and Concepts
* **Angular 19+**: Use of Standalone Components and the new Control Flow (`@if`, `@for`, etc.).
* **Reactive State Management**: Implementation of **Signals** for efficient and fine-grained state management.
* **Reactive Programming**: Extensive use of **RxJS**. For example, managing authentication flows using `user$` in global services.
* **Architecture**: Separation of responsibilities, Guards, Interceptors, and Lazy Loading.
* **Design**: Integration with Tailwind CSS / Bootstrap and animation management.
